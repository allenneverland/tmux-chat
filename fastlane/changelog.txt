Flag Day architecture migration / Flag Day アーキテクチャ移行

Breaking changes:
- QR onboarding removed / QRオンボーディング廃止
- Cloudflare Access Service Token compatibility removed / Service Token互換廃止
- Open mode removed; bearer token required / open mode廃止、Bearer必須

New baseline:
- In-app SSH onboarding with host-agent install/pairing / アプリ内SSHオンボーディング
- tmux bell -> host-agent -> push-server -> APNs notification path / 通知経路刷新
- Claude/Codex hooks compatibility preserved via `reattachd notify` / hooks互換維持
